# Kisaan_Sathi_AI_Presentation
An intelligent, WhatsApp-based agricultural advisory system designed for Indian farmers. It provides instant, dialect-aware support in 12 languages, including features like AI-driven crop disease diagnosis, hyperlocal weather forecasts, mandi price tracking, and government scheme navigation, all without requiring a dedicated mobile app.

Kisaan Sathi — AI Farm Advisor for IndiaKisaan Sathi is an intelligent, WhatsApp-based agricultural advisory system designed to bridge the information gap for India's farmers. It provides instant, dialect-aware support in 12 languages, ensuring that expert agricultural advice is accessible to everyone, regardless of internet speed or technical literacy.  

Key FeaturesVoice- 

1) First, Dialect-aware Interaction: Supports interaction in 12 Indian languages, including regional dialects like Bhojpuri and Maithili.

2) AI Crop Diagnosis: Users can snap a photo of a sick leaf or pest, and the AI will identify the disease and recommend treatment.

3) Hyperlocal Insights: Real-time integration with the Agmarknet API for mandi prices and hyperlocal weather forecasting.

4) Government Scheme Navigator: Identifies eligible schemes (e.g., PM-Kisan, PMFBY) based on land size and crop, and guides users through the application process.

5) Zero-Friction Access: Operates directly via WhatsApp—no new app, no registration, and no OTP required.



# How It Works: The Scoring Engine

The core of Kisaan Sathi is a heuristics-based scoring engine that provides consolidated advice.  

1) Data Retrieval: Integrates real-time data from the Agmarknet API and hyperlocal weather forecasts.

2) Scoring Engine: Evaluates and ranks treatments based on three primary factors:

   Cost: Ensuring the treatment is affordable for the farmer.
   Locality: Matching recommendations with local availability.
   Crop Stage: Providing timing-specific advice (e.g., vegetative stage vs. harvest).

3) Final Decision: Delivers clear, actionable advice in the farmer's preferred language.
